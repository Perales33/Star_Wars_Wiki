Etiquetas HTML Utilizadas:

<nav> - Empleado para poder crear la barra de navegación (Menú)
<a> - Empleado para poder crear enlaces a otras páginas.
<ul> - Empleado para poder crear listas no ordenadas.
<li> - Empleado para añadir elementos a las listas.
<div> - Empleado para poder diferentes partes que contengan contenido.
<img> - Empleado para poder poner imagenes en el footer.
<form> - Empleado para crear un formulario.
<input> - Empleado para campos del formulario como son el nombre y el email.
<textarea> - Empleado en el formulario para pedir la opinion de las personas.
<button> - Empleado para poder ir a otras paginas.

Propiedades CSS Utilizadas:
background-color - Cambia el fondo de pantalla a otro color
font-family - Fuente empleada para los texto.
color - Cambiamos el color de los p y h1 a otro color.
font-size - Empleado para hacer más grande el tamaño de los p.
border-radius - Se aplica para cambiar el radio de los bordes de las imagenes.

Bootstrap:
Bootstrap se empleo para poder dar un estilo más visual y llamativo al formulario, la galeria de imagenes y el formulario.

JavaScript:
JavaScript usado simpleemente para que el formulario envie una alerta para decir que se a enviado el formulario correctamente.
